Observer

Das Projekt im WS 2005/06 sind Programme zum Monitoring der verschiedensten Daten. Festgelegt wurden folgende Eigenschaften:
  • Stand-alone-Server
  • Port 1860
  • Klartext-Protokoll (wie immer)
  • Client-Authentisierung

Datenerfassung:

Statische Daten:
  • Hostname
  • OS-Version (uname, nmap)
  • CPU-Info (/proc/cpuinfo, geladene Module)
  • Speicher-Info (/proc/meminfo)
  • USB-Devices (/proc/bus/usb/devices)
  • Netzwerk-Info (ifconfig, iwconfig, route)
  • Bootmessages (dmesg)
  • Dateisystem, Plattengröße (df, mounti, /etc/[v]fstab)
  • Offene Ports (nmap)

Dynamische Daten:

  • freier Platten-Speicher (df, du)
  • freier Arbeitsspeicher, Load, Anz. User, Uptime (top)
  • Netzwerk-Verkehr (/proc/net/dev, netstat)
  • laufende Prozesse (nur bestimmte, ps-Kommando)
  • User (last-Kommando, who[do]-Kommando)
  • ggf. Anfragen an Standard-Dienste (Monitoring)
